Linux Agent Island Architecture

System Design Overview & Component Interactions

Event Driven

Hooks installed in Gemini, Claude, and Codex CLI push real-time session events to a central backend via Unix Domain Sockets.

Stateful UI

GTK-based X11 island maintains a persistent view of all active agent sessions, communicating with the backend over D-Bus.

Deep Linking

Integrates with X11 utilities (wmctrl/x11-utils) to identify agent terminal windows and perform instant "jumps" from the dashboard.
